# rock-paper-sissors
Odin Project Foundations Javascript chapter solution for "Project: Rock Paper Sissors" and "Revisiting Rock paper sissors.

23/09/2024 : Console-based rock paper sissors game. - https://www.theodinproject.com/lessons/foundations-rock-paper-scissors
             > As per OP learing order, no loops are used.
             
24/09/2024:  Branch created for DOM manipulation UI - https://www.theodinproject.com/lessons/foundations-rock-paper-scissors
             > Three buttons for player options.
             > DOM manipulation used to set round count, round result, scores and final result.
             > Reset game button created when player or computer scores 5 points.
             > Till reset button is pressed the choice buttons are disabled.
